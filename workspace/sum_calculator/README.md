# Sum Calculator

A simple Python project that provides a function to calculate the sum of a list of numbers.

## Features

- Calculate the sum of a list of numbers
- Handles integers and floats
- Includes unit tests

## Installation

```bash
git clone https://github.com/example/sum_calculator.git
cd sum_calculator
pip install -r requirements.txt
```

## Usage

```python
# Example usage
from sum_calculator import calculate_sum

result = calculate_sum([1, 2, 3, 4, 5])
print(result)  # Output: 15
```

## Running Tests

```bash
pytest tests/
```
